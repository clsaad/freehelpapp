Como usar arquivos TSV

Arquivos TSV são planilhas exportadas em formato de texto simples, onde as colunas são separadas por tabulação.
Recomenda-se a edição através das planilhas do Google Drive, pois essa pode exportar a edição também como TSV.

Confira o exemplo na Tabela:

id nome idade
1 João 21
2 Maria 19
3 José 34

*Note que a segunda linha contem informação sobre o tipo de dado armazenado na coluna.

Selecionando o conteudo do arquivo, copiando (Ctrl+C) e colando em uma planilha do Google (Ctrl+V), o resultado será esse:



Para realizar o teste, vamos editar o item com id 3 e incluir mais um, com o id 4, conforme a imagem abaixo:



Faça o Download da planilha no formato TSV, seguindo o caminho Arquivo -> Fazer Download Como -> Arquivo separado por Tabulação (.tsv)



Ao importar os dados TSV utilizando o menu sobre a tabela, o resultado será esse:

id nome idade
1 João 21
2 Maria 19
3 José Matheus 34
4 Pedro 12

Os itens com o mesmo id serão alterados, e novos ids serão incluidos na tabela!
